# 🏦 Bank Management System

A console-based Bank Management System built using Core Java.

## Features
- ✅ User Registration
- ✅ User Login with authentication
- ✅ View all users
- ✅ Delete user
- ✅ Transaction History
- ✅ Date & Time tracking

## Tech Stack
- Java (Core)
- OOP Concepts (Classes, Objects, Inheritance, Encapsulation)
