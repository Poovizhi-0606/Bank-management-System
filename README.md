# 🏦 Bank Management System

A console-based Bank Management System built using Core Java.

## Features
- ✅ User Registration
- ✅ User Login with authentication
- ✅ View all users
- ✅ Delete user
- ✅ Transaction History
- ✅ Date & Time tracking

## Tech Stack
- Java (Core)
- OOP Concepts (Classes, Objects, Inheritance, Encapsulation)

## How to Run
1. Clone the repository
2. Open in Eclipse or VS Code
3. Run `Main.java`
